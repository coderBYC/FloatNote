**Privacy Policy**

This extension respects user privacy and does not collect or transmit personal data to any external servers.

Data Collection
The extension stores user-generated content such as notes, highlights, and interface preferences. This data may include text selected on web pages and notes created by the user.

Data Storage
All data is stored locally in the user’s browser using Chrome’s built-in storage mechanisms (such as chrome.storage or IndexedDB). The data never leaves the user’s device.

Data Usage
The collected data is used solely to provide the core functionality of the extension, including restoring notes, highlights, and layout states on supported web pages.

Data Sharing
The extension does not share, sell, or transfer user data to third parties.

Remote Code
The extension does not load or execute remote code. All scripts and resources are bundled within the extension package.

User Control
Users can delete all stored data at any time by removing the extension or clearing the extension’s storage.

Changes
This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on the extension’s store listing.
